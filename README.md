# Desafio Go - FullCycle 2.0
Este é um projeto de estudo do modulo Docker do curso FullCycle 2.0 da Code Education

**Objetivo:**
  - Criar uma imagem docker que compile um arquivo Go e retorne a mensagem "FullCycle Rocks!!!"
